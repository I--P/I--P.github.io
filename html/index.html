ltisys.controller_hessenberg(A, B, C=None)¶

Reduce the linear system AX + BU to its controller Hessenberg form.

Reduce the linear system AX + BU to the controller Hessenberg form H, Bbar where P*A*P.T = H and P*B = Bbar with P being an orthogonnal similarity transformation.

The new system has the following form:

H is a block upper Hessenberg matrix.

Hc,Bc is the controllable part of the original system and AX + BU is controllable if H is of full rank. If the real part of the eigenvalues of Hnc are negative the system will be stabilizable.

A, B : ndarray
State-space representation of linear system AX + BU.
C : ndarray, optionnal
The output matrix of the linear system (i.e Y = CX). Default is None.
H, Bbar : ndarray
The controller Hessenberg form of the linear system AX + BU
Cbar : ndarray

Optionnaly if C (output matrix) is given the function will also return Cbar = C*P otherwise Cbar = None will be returned. Cbar has the following form:

With Cc such that the transfer function of (A,B,C) and (Hc,Bc,Cc) are the same.

P : ndarray
The orthogonnal similarity transformation
control_index : int
The controllability index, i.e the number of controllable states, of the linear system AX + BU. If control_index==A.shape[0] then the linear system is controllable. Control_index is also the rank of H.

The algorithm comes from [1]. The output from this function has been tested with random controllable and uncontrollable system against Octave which uses tb01ud from Slicot and all results were similar using the default tolerances of allcose.

[1]Datta Biswa, “Numerical Methods for Linear Control Systems”, Elsevier, pp 173-183, 2003.
